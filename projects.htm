Cube Solver:


A backend piece of software capable of solving a rubix cube, developed in Python in 2022

  • takes in a json file with specified colors on each face
  • configures a backend model that translates the color inputs to a functional rubix cube
  • confirms whether the configuration is valid
  • (currently) uses a brute force algorithm in a breadth-first-search style to return the needed rotations to solve the cube due to the exponential nature of the rubix cube configuration tree, the algorithm is only quick (and feasible) for solving scrambles of less than 6 moves

Catan:


A terminal-based version of a classic board game written in OCaml for my CS 3110 final project in 2021

  • randomly generates Catan maps that can keep track of players settlements and roads
  • distributes resources to players based on roll number and settlement location
  • blocks resource collection based on location of robber which changes
  • uses strictly recursion to change game state based on user input

Shortest Path Models:


A series of Jupyter notebooks illustration the application of graph theory to real-life problems developed for ENGRI 1101 in 2021

  • Application of Djikstra's algorithm to minimize the variance of line lengths in a text file with a model that takes the user through each iteration of the algorithm with the corresponding output
  • Application of Djikstra's algorithm to minimize the cost of maintaining an inventory of pipes with a toy example of how a small inventory can be modeled with a graph and how deciding what pipes ot store corresponds to paths in the graph
